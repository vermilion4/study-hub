topic: "Variables & Data Types"
order: 1
cards:
  - q: "What is a variable?"
    a: "A named storage location that holds a value of a particular type."
  - q: "Name three Java primitive types and what they hold."
    a: "int (whole numbers), double (decimals), boolean (true/false); also char (a single character) and long."
  - q: "What's the difference between `int` and `double`?"
    a: "int stores whole numbers; double stores floating-point numbers with a fractional part."
  - q: "Why is `String` not a primitive type in Java?"
    a: "It's a class (object) representing a sequence of characters, with methods like length() and substring()."
  - q: "What does declaring a variable `final` do?"
    a: "Makes it a constant — it can be assigned once and not reassigned."
