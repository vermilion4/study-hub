cards:
  - q: "What do the three parts of a `for` loop header control?"
    a: "Initialization (runs once), the condition (checked before each pass), and the update (runs after each pass)."
  - q: "When would you choose a `while` loop over a `for` loop?"
    a: "When the number of iterations isn't known up front and depends on a runtime condition."
  - q: "What does an `if / else if / else` chain do?"
    a: "Evaluates conditions in order and runs the first matching block; `else` runs if none match."
  - q: "What is the difference between `==` and `.equals()` for Strings in Java?"
    a: "`==` compares references (identity); `.equals()` compares character contents. Use `.equals()` to compare String values."
  - q: "What does `break` do inside a loop?"
    a: "Immediately exits the innermost loop; `continue` skips to the next iteration instead."
