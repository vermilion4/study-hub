cards:
  - q: "What is prompt engineering?"
    a: "The practice of writing and refining inputs so a model produces more useful, reliable output."
  - q: "Name a few techniques that improve prompt results."
    a: "Be specific and give context; provide examples (few-shot); ask the model to reason step by step; specify the desired format."
  - q: "What is 'few-shot' prompting?"
    a: "Including a few worked examples of the task in the prompt so the model follows the pattern."
  - q: "Why ask a model to 'think step by step'?"
    a: "Encouraging intermediate reasoning often improves accuracy on multi-step problems."
  - q: "Why should you verify an AI assistant's code output?"
    a: "It can be confidently wrong or subtly buggy; you own correctness, so read, test, and validate what it produces."
